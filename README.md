# EmoDiary

EmoDiary 안드로이드 앱

</br>

## 프로젝트 소개

일기를 작성하고, 작성한 일기를 분석하여 사용자의 감정과 워드 클라우드로 일기의 내용을 보여주는 안드로이드 앱입니다.

</br>
